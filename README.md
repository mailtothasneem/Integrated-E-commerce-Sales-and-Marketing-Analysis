## 📌 Project Overview
This project focuses on analyzing e-commerce sales, inventory, customer behavior, product returns, and marketing campaign data.  
The analysis was performed using **Microsoft Excel** for preprocessing and **Power BI** for dashboard development.  
The outcome is an interactive dashboard that provides actionable insights for business decision-making.

---

## 🎯 Objectives
- Clean and organize raw business data  
- Improve data quality through preprocessing  
- Prepare datasets for business intelligence reporting  
- Analyze sales, inventory, customer, and marketing performance  
- Develop interactive dashboards using Power BI  
- Generate meaningful insights for decision-making  

---

## 🗂️ Data Sources
- Source: [ExcelX Practice Data](https://excelx.com/practice-data/generators/ecommerce-marketing/)  
- Timeline: 2023–2026  
- Domain: Commerce & Marketing Campaign  

---

## 🛠️ Tools & Technologies
- **Excel**: Data Cleaning, Power Query, Pivot Tables  
- **Power BI**: Data Modeling, DAX, KPIs, Visualizations  

---

## 🔧 Data Preprocessing
- Removed duplicates and handled missing values  
- Standardized formats and created calculated fields  
- Built fact and dimension tables for analysis  
- Applied star schema for optimized dashboard performance  

---

## 📐 Data Modeling & DAX
- Central Fact Table: Order Item Table (transaction-level details)  
- Dimension Tables: Customer, Product, Order, Inventory, Returns, Campaign  
- Relationships created using CustomerID, ProductID, OrderID, CampaignID  

### Key DAX Measures
- Total Sales, Total Orders, Customer Count  
- CTR %, Conversion Rate %, Cost Per Click  
- Return Orders, Stock Status, Avg Unit Price  

---

## 📊 Dashboard Visuals
The Power BI dashboard includes seven visuals covering:
- Sales performance  
- Customer growth and city distribution  
- Product category analysis  
- Inventory and warehouse stock status  
- Return orders by reason  
- Marketing campaign effectiveness  
- Predictive analysis (forecasted sales and customer growth)  

---

## 📈 Key Insights
- Total Sales: ₹279.42K with 1K orders and 800 customers  
- Top Cities: Surat, Mumbai, Bangalore, Ahmedabad  
- Order Status: Delivered (195) and Processing (214) dominate  
- Product Categories: Apparel and Electronics lead sales  
- Marketing: Affiliate channel generated highest conversions (515)  
- Returns: Low return rates indicate strong product quality  

---

## 🔮 Predictions
- Sales Growth: Expected rise of 10–15% next quarter  
- Customer Expansion: Likely to cross 1,000+ customers by 2026  
- Top Categories: Electronics and Apparel remain dominant  
- Emerging Potential: Beauty & Health category

## ✅ Recommendations
